P0-001 Webhook 处理中存在严重的全表扫描性能问题

定位：src/modules/webhook/webhook.service.ts:57

分类：performance

影响：每次 Webhook 回调都会查询所有任务记录并在内存中过滤，当任务数量增长到数千条时会导致严重的性能问题和数据库压力，可能造成 Webhook 超时失败

实际：调用 findAll 获取所有任务，然后在内存中用 find 过滤

修复建议：

在 DiagnosisTaskService 中添加 findByOceanTaskId 方法：

async findByOceanTaskId(oceanTaskId: string): Promise<DiagnosisTask | null> {
  return this.taskRepository.findOne({
    where: { oceanTaskId },
    relations: ['material', 'config'],
  });
}

然后在 webhook.service.ts 中替换：
const task = await this.taskService.findByOceanTaskId(task_id);

P1高优先级问题（应在合并前修复）

P1-001 Webhook 签名验证缺少时间戳验证，存在重放攻击风险

定位：src/modules/webhook/webhook.service.ts:28

分类：security

影响：攻击者可以捕获合法的 Webhook 请求并无限次重放，导致任务状态被恶意修改

期望：应该验证时间戳在合理范围内（如 5 分钟），并使用 Redis 记录已处理的请求 ID 防止重放

实际：只验证签名正确性，不验证时间戳新鲜度

修复建议：

在 verifySignature 方法中添加时间戳验证：

verifySignature(timestamp: string, body: string, signature: string): boolean {
  // 验证时间戳（5 分钟内有效）
  const requestTime = parseInt(timestamp);
  const now = Date.now();
  if (Math.abs(now - requestTime) > 300000) {
    this.logger.warn(`时间戳过期: ${timestamp}`);
    return false;
  }

  const expectedSignature = crypto
    .createHmac('sha256', this.webhookSecret)
    .update(timestamp + body)
    .digest('hex');

  return crypto.timingSafeEqual(
    Buffer.from(expectedSignature),
    Buffer.from(signature)
  );
}

P1-002 Token 刷新逻辑缺少并发控制，可能导致多次刷新

定位：src/modules/ocean-engine/ocean-engine.service.ts:72

分类：correctness

影响：当多个请求同时发现 token 过期时，会并发调用 refreshAccessToken，导致不必要的 API 调用和可能的 refresh_token 失效

期望：使用分布式锁确保同一时间只有一个请求在刷新 token

实际：没有并发控制，多个请求会同时调用刷新接口

修复建议：

使用 Redis 分布式锁：

private async getAccessToken(advertiserId: string): Promise<string> {
  const cacheKey = `ocean_engine:token:${advertiserId}`;
  let token = await this.redis.get(cacheKey);

  if (!token) {
    const lockKey = `ocean_engine:token_lock:${advertiserId}`;
    const lockValue = Date.now().toString();

    // 尝试获取锁（10 秒超时）
    const locked = await this.redis.set(lockKey, lockValue, 'EX', 10, 'NX');

    if (locked) {
      try {
        token = await this.refreshAccessToken(advertiserId);
      } finally {
        await this.redis.del(lockKey);
      }
    } else {
      // 等待其他请求刷新完成
      await new Promise(resolve => setTimeout(resolve, 100));
      token = await this.redis.get(cacheKey);
      if (!token) {
        throw new Error('Token 刷新失败');
      }
    }
  }

  return token;
}
